# Dodgeball

This is a simple browser-based, two-player dodgeball game built as a fun side project. The goal is to hit the other player with a ball while trying to dodge their balls.

## Get Started

```bash
# 1. Clone the repository
git clone https://github.com/myang35/dodgeball.git

# 2. Navigate to the project directory
cd dodgeball

# 3. Install dependencies
npm install

# 4. Build the project
npm run build

# 5. Run the build files
npm run preview
```

## Controls

### Player 1

Move: Arrow keys

Throw: Space

### Player 2

Move: WASD

Throw: Enter
